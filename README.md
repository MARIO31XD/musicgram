Project about mix of Spotify and Instagram in a single app.

#Musicgram

-Objetivos básicos

1. Pantalla de inicio donde poder Registrarse (con google) o iniciar sesión.
2. Escuchar canciones.
3.Crear Playlists
4. Compartir PlayLists
5. Seguir usuarios

-Objetivos a largo plazo

6.Dar like a Playlists
7.Comentar PlayLists
8. Ver un feed de actividad
9. IA generativa de PlayLists según tus gustos musicales
